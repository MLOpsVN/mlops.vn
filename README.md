# mlops.vn

## Quickstart

Node.js **v17.9.0**

```bash
npm install
npm start
npm run build
```

## Troubleshooting

### Custom domain

When using custom domain with Github Pages, we need to set `homepage` field in `package.json` to `https://MLOpsVN.github.io` instead of `https://MLOpsVN.github.io/mlops.vn`
